ANIMATION & RIGGING COURSE OUTLINES

Beginner Topics

Introduction to Animation (CODE: AN101)

Learn the basics of animation, including keyframes, timing, and storytelling in motion graphics.

2D Animation Fundamentals (CODE: AN102)

Explore traditional and digital 2D animation techniques using tools like Adobe Animate.

Intermediate Topics

Character Animation (CODE: AN201)

Dive into character rigging, facial expressions, and dynamic movements to bring characters to life.

Motion Graphics (CODE: AN202)

Create visually engaging motion graphics for advertising, branding, and storytelling.

Storyboarding Techniques (CODE: AN203)

Learn how to plan and visualize animations with effective storyboarding methods.

Advanced Topics

3D Animation Basics (CODE: AN301)

Explore 3D modeling, rigging, and animation using tools like Prisma3D or FlipaClip.

Special Effects Animation (CODE: AN302)

Add visual effects like explosions, particles, and lighting to animations for a cinematic feel.

Advanced Animation Projects (CODE: AN303)

Work on real-world projects to create animations for games, movies, or interactive media.
